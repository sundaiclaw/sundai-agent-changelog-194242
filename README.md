# AgentChangelog

AgentChangelog converts messy AI-agent run logs into a clean changelog and incident summary.

## What it does
- Input: raw run logs / notes
- Output: wins, failures, root-cause, and next-action changelog

## How to Run (from zero)
1. Prerequisite: Node.js 18+
2. `git clone https://github.com/sundaiclaw/sundai-agent-changelog-194242`
3. `cd sundai-agent-changelog`
4. `npm install`
5. `npm start`
6. Open `http://localhost:3000`
